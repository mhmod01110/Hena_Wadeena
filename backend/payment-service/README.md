# Hena Wadeena Payment Service

- Port: `8007`
- Database env: `PAYMENT_DATABASE_URL`
- API prefixes: `/api/v1/payments`, `/api/v1/wallet`
- Resource path: `/transactions`

Endpoints (for each prefix):
- `POST /transactions`
- `GET /transactions`
- `GET /transactions/{id}`
- `PATCH /transactions/{id}`
